{% block title %} About Timer Cube {% endblock %}

Time tracking made easy. Build yourself a high-tech dice, with as many faces as you please, and easily setup each face to track individual tasks. Integrate with external DevOps tools to report task spend/effort and increase client confidence with accurate spend reports.

Key Features

  • Self-Hosted - Designed to run on containers, giving you full control of deployment
  • Expandable - As many dice, dice faces, tasks and task types as you desire
  • Integration - Connect to 3rd party DevOps/Project Management Systems
  • FOSS - Build upon this system as you please, make your ideas a reality
  • Low Power - Dice designed to run on battery for many days without charge
